# Beauty hair

Эту задачу следует реализовать с помощью фреймворка Flask. Серверной частью БД будет PostgreSQL.

## Task

Для клиента:

- Приложение должно иметь форму регистрации с полями "Имя", "Фамилия", "Номер телефона", "Почта".

- Приложение должно иметь форму аутентификация и авторизации (права доступа владелец/арендатор/субарендатор). При успешной аутентификации пользователь переходит в личный профиль, в котором отображена вся его история посещений салона (мастера). Там же должна быть продублирована кнопка "Записаться".

- В приложении должен быть перечень услуг с актуальной стоимостью и продолжительностью по времени.

- Приложение должно быть портфолио работ мастера.

- На главной странице должна иметься кнопка "Записаться". При нажатии на которую отображается календарь (на месяц или более) с отображением свободного времени для записи. Нажимая на свободное время должно появляться выпадающее меню с перечнем услуг. При этом, если желаемая услуга не вписывается по времени в свободное "окно", то клиент должен быть оповещён flash-сообщением. Так же должна быть возможность отменить запись.

- В приложении должна быть страница отзывов с полем, в котором пользователь имеет возможность оставить свой отзыв.

- В приложении должна быть страница с контактами данного мастера/салона.

Для мастера:

- Список клиентов. При нажатии на имя клиента, будет совершён переход на его историю посещений.

- Расписание в виде календаря. При нажатии на выбранный день, открывается этот день с возможностью записывать клиента в любое время с аналогичным выпадающим меню. При этом, если желаемая услуга не вписывается по времени в свободное "окно", то мастер должен быть оповещён flash-сообщением. Так же должна быть возможность отменить запись.

 - Оплаты. Оплата может осуществляться несколькими путями:

 а) оплата наличными;
 б) оплата картой (следует реализовать логику для возможности оплаты через терминал);
 в) оплата в долг (кнопка, нажимая которую, в историю заказа добавляется информация что работа выполнена, но ожидается оплата);
 г) чаевые.

 - Расходные материалы. Должны иметься поле, в котором записывается купленные расходные материалы. Поле, в котором записываются проданные косметические средства.
